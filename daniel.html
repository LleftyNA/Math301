The Layer method

1:Tips corners

Turn tips corners to match adjacent center block. (This step actually doesn't matter, because tips move independently. But it will be easier to see.)

2:Match yellow(or bottom face) center block

By observing color of tips corners and adjacent center blocks, we turn the three center blocks to a same face. Sometime it requires a little time to solve the face which three center blocks can all turn to the face.

3:Solving bottom face and layer(need new figure)

Now we need to solve the yellow (bottom) face and adjacent layer.
We already match tips corners and center blocks. So there are three edges blocks which we need to solve and we don't care about upper layer for now. There are two situations:

First : the required edge block is in upper layer, then we just need to turn the block to match the center color. Here, if the edge block is at left side, then we are using left side algorithm: R U' R. If it is at right side, then, we are using: L' U L.

Second : the required edge block is at bottom layer, then we need to move it to upper layer first by:R U/U' R'. We don't care which block will be move down, the only thing we care is moving the required block to upper layer, so we can use first step to move it to the right place.

4:Final layer and situations(need new figure)

In this part, we need to solve the remainning edges without moving the bottom face and layer.
Let's match the upper layer center with bottom solved layer. Then, there will be three situations:
First double edges flip: L R' L' R (There is one more cycle, need double check)
Second edges face double-three cycle: moving face to left:R U R' U R U R' to right:R U' R' U' R U' R'
Third triple edges swap: if the solved lay is at left:R' U' L' U L R at right:L U R U' R' L'
